> **Nota:** Version de PHP 8.2.4
**Nota:** Version MariaDb o Mysql= 10.4.28-MariaDB

Para la ejecucion del proyecto se quiere xaampp.

1. El proyecto se debe clonar en la carpeta de xampp -> htdocs 
2. http://localhost/phpmyadmin/ Ingresas a esta URL y creas una base de datos que se llame prueba_suplos
3. Una vez creada la base de datos en el encarpetado se encuentra el archivo .sql para importarlo. estando en esta url con la base de datos creada http://localhost/phpmyadmin/ -> Dirigir a la opcion importar en la parte superior -> selecioonas el archivo .sql -> boton continuar
4. Una vex se importe las tablas puedes acceder al link para la visualizacion y navegacion del proyecto: http://localhost/prueba/
5. Proyecto MVC
6. Con frameworks como Bootstrap, Jquery y xls. full para archivos de excel.
7. En el reporte de excel se crean 2 hojas con diferentes tablas
